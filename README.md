# Decision Tree Model - Loan Campaign

This project implements a decision tree model for analyzing loan campaign data.

## Files
- `AIML_ML_Project_full_code_notebook.ipynb` - Main Jupyter notebook with the model implementation
- `Loan_Modelling.csv` - Dataset used for training and analysis

## Setup
1. Clone the repository
2. Install required dependencies (see notebook for specific packages)
3. Open the Jupyter notebook to run the analysis

## Usage
Run the notebook cells sequentially to train and evaluate the decision tree model.
